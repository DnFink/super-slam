# Super Slam

Welcome to **Super Slam**, a web-based action/platformer game built with HTML, CSS, and Vanilla JavaScript!

## Features

- **Unique Characters**: Play as different characters, each with their own special abilities and playstyles.
- **Two-Life System**: A forgiving yet challenging life system to keep the gameplay engaging.
- **Dynamic Levels**: Navigate through exciting levels filled with obstacles and enemies (including Bullet Bills!).
- **Custom Audio & Assets**: Integrated sound effects and visuals.

## How to Play

1. Clone or download this repository.
2. Open `index.html` in your favorite modern web browser.
3. Select your character and jump into the action!

## Project Structure

- `index.html` / `index.css`: The main entry point and styling for the game UI and menus.
- `game.js`: The core game engine and logic.
- `audio.js`: Handles game sounds and music.
- `assets.js`: Manages game assets (images, sprites).
- `levels.txt` / `gen.js`: Level definitions and generation logic.
